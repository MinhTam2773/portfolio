# AllBarber Gallery Images

This directory should contain gallery images for the AllBarber project.

## Required Images:

1. **dashboard.png** - Screenshot of the main barbershop dashboard showing queue status and analytics
2. **queue-management.png** - Interface for managing customer queues in real-time
3. **admin-panel.png** - Admin panel showing multi-tenant management and settings
4. **customer-view.png** - Customer-facing interface for joining queues and viewing wait times

## Image Specifications:
- Format: PNG or JPEG
- Recommended size: 1920x1080 (landscape orientation)
- Max file size: 500KB each (optimize for web performance)
- Aspect ratio: 16:9 preferred for consistent display

## Tips:
- Use high-quality screenshots from the actual application
- Ensure all screenshots have clean, professional appearance
- Remove any sensitive/personal data before adding images
- Consider adding subtle shadows or borders for visual polish
