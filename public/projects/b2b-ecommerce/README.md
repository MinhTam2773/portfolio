# B2B Ecommerce Platform Gallery Images

This directory should contain gallery images for the B2B Ecommerce platform project.

## Required Images:

1. **product-catalog.png** - Product catalog interface with bulk ordering capabilities
2. **bulk-order.png** - CSV upload interface and bulk order management screen
3. **approval-workflow.png** - Order approval workflow dashboard showing pending/approved orders
4. **admin-dashboard.png** - Admin panel with customer management, pricing tiers, and analytics

## Image Specifications:
- Format: PNG or JPEG
- Recommended size: 1920x1080 (landscape orientation)
- Max file size: 500KB each (optimize for web performance)
- Aspect ratio: 16:9 preferred for consistent display

## Tips:
- Emphasize B2B-specific features (bulk ordering, approval workflows)
- Show customer-specific pricing and volume discounts
- Highlight admin capabilities for managing multiple customers
- Use professional product images and realistic business data
