# Pathr Gallery Images

This directory should contain gallery images for the Pathr career advisor project.

## Required Images:

1. **career-path.png** - Visual representation of generated career paths and roadmaps
2. **skill-analysis.png** - Skill gap analysis interface showing current vs. target competencies
3. **resources.png** - Learning resources and course recommendations page
4. **progress-tracking.png** - User progress dashboard with milestones and achievements

## Image Specifications:
- Format: PNG or JPEG
- Recommended size: 1920x1080 (landscape orientation)
- Max file size: 500KB each (optimize for web performance)
- Aspect ratio: 16:9 preferred for consistent display

## Tips:
- Showcase the interactive career path visualization
- Highlight the skill gap analysis charts/graphs
- Include examples of personalized recommendations
- Use mock data if real user data is sensitive
