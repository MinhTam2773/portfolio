# Sharie Gallery Images

This directory should contain gallery images for the Sharie audio-sharing platform.

## Required Images:

1. **feed.png** - Main feed showing audio posts from followed artists/creators
2. **upload.png** - Audio upload interface with metadata input and waveform visualization
3. **profile.png** - User profile page showing uploaded tracks and follower network
4. **player.png** - Audio player interface with playback controls and social interactions

## Image Specifications:
- Format: PNG or JPEG
- Recommended size: 1920x1080 (landscape orientation)
- Max file size: 500KB each (optimize for web performance)
- Aspect ratio: 16:9 preferred for consistent display

## Tips:
- Showcase the audio waveform visualizations
- Highlight social features (likes, comments, shares)
- Display the clean, modern UI with audio player controls
- Use mock audio tracks if real content is copyrighted
